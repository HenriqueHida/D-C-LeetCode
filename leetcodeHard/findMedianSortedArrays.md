# Solução LeetCode: findMedianSortedArrays

Este repositório contém uma solução para o problema "findMedianSortedArrays" do LeetCode. O script inclui:

- Uma função `findMedianSortedArrays` que implementa a solução.
- Manipulação de entradas.
- Medição do tempo de execução.

## Descrição do Problema

Dadas duas arrays ordenadas `nums1` e `nums2` de tamanho `m` e `n` respectivamente, retorne a mediana das duas arrays ordenadas.

A complexidade geral de tempo de execução deve ser `O(log(m+n))`.

### Exemplo 1:

Input: nums1 = [1,3], nums2 = [2]
Output: 2.00000
